# Minecraft Mace Auto-Farm 我的世界全自动重锤农场

## Description 说明

本项目通过自动化脚本实现不详药水投喂、假人批量生成和管理，用来批量生产（不详）试炼钥匙。

This project uses automated scripts to implement the feeding of unknown potions, the batch generation and management of dummies, and to mass-produce **(ominous) trial keys**.

## 前置条件

- 首先用轰炸机清空试炼之地的所有非功能性方块，围绕刷怪笼 4 * 2 * 4 的区域建造怪物处死装置，并在刷怪笼 14 格内搭建假人放置处。First, use bombers to clear all non-functional blocks in the Trial Grounds. Then, build a monster execution device around the 4 * 2 * 4 area of the mob spawner, and place a dummy within 14 blocks of the mob spawner.

- 已经安装了 Node.js (版本号>=18)。Node.js (version>=18) is already installed.

## 运行

```bash
npm run start
```
